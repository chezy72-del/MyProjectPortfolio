Project Overview

This is a C++ implementation of the classic game Rock Paper Scissors Lizard Spock.

The game allows a user to play against the computer, which makes random choices.

The concept of this game came from The Big Bang Theory, please watch the below clip to get the rules of Rock Paper Scissors Lizard Spock.

Features

  • Randomized computer moves
  • Score tracking for both player and computer
  • Input validation to ensure valid moves
  • Clear instructions and feedback after each round
