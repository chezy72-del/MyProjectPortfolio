Traffic Safety Data Analysis Project

This project involves analyzing traffic safety data to identify patterns and trends.

The goal is to use Python to process and visualize the data, providing insights on if there is a correlation between smartphone usage and traffic accidents.

The first dataset can be found on the National Highway Traffic Safety Administration’s website.

The second dataset can be found on the Pew Research Center’s website.
